---------------------
Instrucciones de uso:
    1- Clonar el repositorio
    2- Añadir el JAR de los drivers de SQLite a las Libraries en NetBeans
    3- Añadir la libreria de Absolute Layout de NetBeans
    4- Correr el proyecto
